# Layout

Control element dimensions, flexbox, grid, and spacing.

## Layout Types

Select the layout type for an element:

| Type | Description | CSS |
|------|-------------|-----|
| **Default** | Block layout | `display: block` |
| **Row** | Horizontal flex | `display: flex; flex-direction: row` |
| **Column** | Vertical flex | `display: flex; flex-direction: column` |
| **Grid** | CSS Grid | `display: grid` |

</Callout>

## Dimensions

Control width and height:

| Property | Description |
|----------|-------------|
| **Width** | Element width in px or `auto` |
| **Height** | Element height in px or `auto` |
| **Aspect Ratio** | Lock width/height ratio |

### Auto Sizing

Set width or height to `Auto` for content-hugging behavior:
- Width auto: Element shrinks to fit content
- Height auto: Element grows with content

## Flex Controls

When Row or Column layout is selected:

### Gap

Space between child elements. Enter a pixel value.

## Padding

Inner spacing (space between element border and content):

| Control | Description |
|---------|-------------|
| **Unified** | Same padding all sides |
| **Individual** | Top, Right, Bottom, Left |
| **Horizontal/Vertical** | X-axis and Y-axis shortcuts |

Enter values in pixels. Use arrow keys for fine-tuning:
- Arrow Up/Down: Вұ1px
- Shift + Arrow: Вұ10px

## Clip Content

Toggle whether content overflows or is clipped:

| Setting | CSS | Effect |
|---------|-----|--------|
| On | `overflow: hidden` | Content clipped |
| Off | `overflow: visible` | Content can overflow |

## Margin

External spacing. See the dedicated Margin section in the right sidebar.

| Control | Description |
|---------|-------------|
| **Linked** | Same margin all sides |
| **Unlinked** | Independent Top, Right, Bottom, Left |

## Tips

1. **Use Row/Column** for most layouts вҖ” flexbox is more predictable
2. **Start with auto sizing** and add fixed dimensions only when needed
3. **Use gap** instead of margins between flex children
4. **Clip content** when elements might overflow unexpectedly
