# HyperIDE Documentation

Welcome to **HyperIDE** — a visual React component editor with AI superpowers.

## What is HyperIDE?

HyperIDE lets you edit React components visually, like Figma, while AI generates clean, production-ready code. It supports Next.js, Vite, Remix, and other React frameworks.

## Key Features

- **Visual Editing** — Edit React components with a Figma-like interface
- **AI Assistant** — Generate and modify code through natural language chat
- **Live Preview** — See changes instantly with Hot Module Replacement
- **Framework Support** — Works with Next.js, Vite, Remix, Create React App, and Bun
- **Docker Isolation** — Your projects run in secure, sandboxed containers

## Quick Start

1. **Clone the repository**
   ```bash
   git clone git@github.com:hyperide/hypercanvas.git
   cd hypercanvas
   ```

2. **Install dependencies and start**
   ```bash
   bun install
   bun run dev
   ```

3. **Open in browser**
   Navigate to `http://localhost:8080` and connect your first project.
